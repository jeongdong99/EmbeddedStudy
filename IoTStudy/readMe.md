1. 문제 정의
  무분별한 쓰레기 불법 투기와 그로 인한 환경오염을 예방하기 위해 
  불법 투기를 단속하는 동시에 사람들에게 유용한 정보를 줄 수 있는
  전광판 역할을 하는 시스템

2. 제안서 
   시스템의 주요 기능은 다음과 같다.
    - 현재 시각, 온도, 습도, 불쾌지수, 날씨를 출력한다.
    - LED를 통해 불쾌 지수를 알린다.
    - 모션이 감지되면 카메라로 영상을 찍어 저장한다. 

3. H/W 또는 시스템 구성

개발 컴퓨터 운영체제 : RaspberryPi
개발 도구 : python
개발 언어 : python
